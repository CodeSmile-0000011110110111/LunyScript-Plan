- => **ScriptRunner.OnStartup**
	- => **ScriptRegistry.DiscoverScripts**
		- find scripts by type
		- creates **ScriptDefinition** (ID) for each type
	- => **ScriptActivator.RegisterObjects(sceneObjects)**
		- takes a list of objects (on startup: all scene objects)
		- creates **ScriptContext** for objects with names matching script type
		- => **ScriptContextRegistry.CreateContext(scriptDef, sceneObject)**
			- keeps the context stored in collection
		- => **LifecycleManager.Register(context)**
			- adds event handlers to context object's events (OnEnable, OnDestroy, ..)
	- => ScriptActivator.BuildScripts(all contexts)
		- calls **LunyScript.Build()** (instance initialized with script context)
